Sprite origins

Section titled â€œSprite originsâ€

Notice how the gizmo is in the center of the sprite. The sprite modifier has an alternative way to achieve what we want, by offseting the visuals instead. This is called the sprite origin, and changes where the transform gizmo / origin is.

The values for a sprite origin range from 0 to 1.

  • On the horizontal (x) axis, 0 is left, 0.5 is center, and 1 is right.
  • On the vertical (y) axis, 0 is bottom, 0.5 is center, and 1 is top.

If we change the y value of the origin to 1, we can set the Y position on the Transform back to 0. This makes the top of our pillar align with the red axis as well!

The Transform gizmo is now also on the top edge of the sprite, not the middle. Itâ€™s always situational which you use, but for our case we can use this.
